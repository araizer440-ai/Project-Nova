# Project Nova

A basic game setup template for building games with Python and Pygame.

## Features

- Basic game loop (60 FPS)
- Player character with movement controls
- Screen boundary collision
- Modular code structure

## Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Running the Game

```bash
python main.py
```

## Controls

- **Arrow Keys** or **WASD** - Move the player
- **ESC** - Exit the game

## Project Structure

- `main.py` - Entry point and initialization
- `game.py` - Main Game class with game loop
- `player.py` - Player character class
- `requirements.txt` - Project dependencies
